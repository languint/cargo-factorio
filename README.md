<div align="center">
    <h1>factorio-rs</h1>
    <p>Rust SDK for Factorio's modding API.</p>
</div>

> [!NOTE]
> This project is in development, expect breaking changes.

## Quick start

```bash
cargo install factorio-rs-cli
mkdir my-mod && cd my-mod
factorio-rs init --name my-mod
factorio-rs build
```
